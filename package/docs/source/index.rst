Vegphenotyping Documentation
===================================

**Vegphenome** is a Python library that provides state of the art alogirthms for phenotyping. It divides the phenotyping in base phenotyping 
class whcih is crop agnostic and each speicifc crop can inherit and modify its behavour. In this way offers a *simple* and *intuitive* API to use and further extend this library for new crops.
It provides serial and mutitask networks for vege phenotyping.\
